# Select

## Overview

Select is a headless compound primitive for single-select dropdowns. It matches Radix UI's API: Root, Trigger, Portal, Content, and Item. Supports keyboard navigation, type-to-search, focus trap, and form integration.

**Anatomy:** Root → Trigger, Portal → Content → Item(s).

## Accessibility

### Keyboard

- **Enter** / **Space** — Open/close; select focused item when open
- **Arrow Down** — Open if closed; move focus to next item when open
- **Arrow Up** — Move focus to previous item when open
- **Home** — Focus first item when open
- **End** — Focus last item when open
- **Escape** — Close, return focus to trigger
- **Typeahead** — Jump to item by typing first letters
- **Tab** / **Shift+Tab** — Close and move focus when open

### Focus

- Focus moves to first item when content opens
- Focus returns to trigger when content closes
- Focus trap inside content when open
- Disabled items are skipped

### Screen Reader

- Trigger: `aria-haspopup="listbox"`, `aria-expanded`, `aria-controls`
- Content: `role="listbox"`; Items: `role="option"`, `aria-selected`
- `data-highlighted` for current focus
- Accessible name from visible text or `aria-label`

### Consumer Responsibilities

You must provide:

- Visible focus indicators
- Positioning (CSS, Popper, Floating UI)
- Display of selected value in trigger (this library does not render the selected label)

### Controlled

```tsx
const [value, setValue] = useState('apple');

<Select.Root value={value} onValueChange={setValue}>
  <Select.Trigger>{value || 'Select'}</Select.Trigger>
  <Select.Portal>
    <Select.Content>
      <Select.Item value="apple">Apple</Select.Item>
      <Select.Item value="banana">Banana</Select.Item>
    </Select.Content>
  </Select.Portal>
</Select.Root>
```

### Displaying Selected Value

The consumer must render the selected label in the trigger. Use controlled mode and map value to label:

```tsx
const options = [{ value: 'apple', label: 'Apple' }, { value: 'banana', label: 'Banana' }];
const [value, setValue] = useState('apple');
const selectedLabel = options.find(o => o.value === value)?.label ?? 'Select';

<Select.Root value={value} onValueChange={setValue}>
  <Select.Trigger>{selectedLabel}</Select.Trigger>
  <Select.Portal>
    <Select.Content>
      {options.map(o => (
        <Select.Item key={o.value} value={o.value}>{o.label}</Select.Item>
      ))}
    </Select.Content>
  </Select.Portal>
</Select.Root>
```

## Edge Cases

- **forceMount:** When true, Content stays in DOM when closed. Use `aria-hidden` and `hidden` for accessibility.
- **loop:** When true, Arrow Down on last item focuses first; Arrow Up on first focuses last.
- **Positioning:** This library is headless. Use CSS, Popper, or Floating UI for positioning.
- **Form submission:** Use a hidden input if you need to submit the selected value; the trigger is a button and does not submit by default.
- **Disabled items:** Skipped in keyboard navigation; not selectable.
