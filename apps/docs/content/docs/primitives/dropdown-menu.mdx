# Dropdown Menu

## Overview

Dropdown Menu is a headless compound primitive for menus triggered by a button. It matches Radix UI's API: Root, Trigger, Portal, Content, Item, Group, Label, Separator. Supports roving tabindex, typeahead, and keyboard navigation.

**Anatomy:** Root → Trigger, Portal → Content → Item(s), Group, Label, Separator.

## Accessibility

### Keyboard

- **Enter** / **Space** — Activate focused item
- **Arrow Down** — Move focus to next item
- **Arrow Up** — Move focus to previous item
- **Home** — Focus first item
- **End** — Focus last item
- **Escape** — Close menu, return focus to trigger
- **Typeahead** — Jump to item by typing first letters
- **Tab** / **Shift+Tab** — Trapped when modal

### Focus

- Roving tabindex: only the focused item has `tabIndex={0}`; others have `tabIndex={-1}`
- Focus moves to first item when menu opens
- Focus returns to trigger when menu closes
- Disabled items are skipped

### Screen Reader

- Content: `role="menu"`; items: `role="menuitem"`
- `data-highlighted` for current focus
- `data-disabled` for disabled items
- Accessible name from visible text or `aria-label`

### Consumer Responsibilities

You must provide:

- Visible focus indicators
- Positioning (CSS, Popper, Floating UI)
- Menu item styling

## Edge Cases

- **forceMount:** When true, Content stays in DOM when closed. Use `aria-hidden` and `hidden` for accessibility.
- **loop:** When true, Arrow Down on last item focuses first; Arrow Up on first focuses last.
- **dir="rtl":** Full RTL support for keyboard navigation.
- **Disabled items:** Skipped in keyboard navigation; not selectable.
- **Typeahead:** Type first letters to jump to matching item; resets after timeout.
