# Headless Primitives

A headless TypeScript library providing UI primitives with full modern accessibility support.

## What is this?

This library delivers unstyled, accessible building blocks that follow Radix UI's API conventions. Primitives handle semantics, keyboard interaction, focus management, and ARIA—consumers provide all styling.

## Key Features

- **WCAG 2.2 AA** — Full accessibility compliance
- **Radix-style API** — Familiar props (`open`, `defaultOpen`, `value`, `checked`, `asChild`, etc.)
- **No visual styles** — Headless only; you control appearance
- **TypeScript** — Full type safety

## Quick Start

<Steps>
  <Step>

  <Step>
